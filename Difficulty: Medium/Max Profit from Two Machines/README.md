The profit earned for performing each task is given in two arrays <strong>a[]</strong> and <strong>b[]</strong> such that if Machine A performs the i-th task, the profit is <strong>a[i]</strong>, and if Machine B performs it, the profit is <strong>b[i]</strong>. </span></p>
<p data-pm-slice="0 0 []"><span style="font-size: 14pt;">Machine A can process at most <strong>x</strong> tasks, and Machine B can process at most <strong>y</strong> tasks. It is guaranteed that <strong>x + y ≥ n</strong>, so all tasks can be assigned. Return the maximum possible profit after assigning each task to either Machine A or Machine B.</span></p>
