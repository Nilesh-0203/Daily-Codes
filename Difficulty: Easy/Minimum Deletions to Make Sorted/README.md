<h2><a href="https://www.geeksforgeeks.org/problems/minimum-number-of-deletions-to-make-a-sorted-sequence3248/1">Minimum Deletions to Make Sorted</a></h2><h3>Difficulty Level : Difficulty: Easy</h3><hr><div class="problems_problem_content__Xm_eO" style="--text-color: var(--problem-text-color);"><p><span style="font-size: 18px;">Given an array <strong>arr[]</strong>, find the minimum number of elements to delete so that the remaining elements form a strictly increasing sequence in the same order.</span></p>
<p><span style="font-size: 18px;"><strong>Examples:</strong></span></p>
<pre><span style="font-size: 18px;"><strong>Input: </strong>arr[] = [5, 6, 1, 7, 4]<strong>
Output: </strong>2<strong>
Explanation: </strong>Removing 1 and 4 leaves [5, 6, 7] which is strictly increasing.</span></pre>
<pre><span style="font-size: 18px;"><span style="font-size: 18px;"><strong>Input: </strong>arr[] = [1, 1, 1]<strong>
Output: </strong>2<strong>
Explanation: </strong>Removing any 2 elements leaves [1] which is strictly increasing.</span></span>
</pre>
