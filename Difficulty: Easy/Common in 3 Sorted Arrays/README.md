If there are no such elements return an empty array.<br><strong>Note:&nbsp;</strong></span><span style="color: #273239; font-family: Nunito, sans-serif;"><span style="font-size: 18px; letter-spacing: 0.162px;">Ignore duplicates, include each common element only once in the output.</span></span></p>
<p><strong><span style="font-size: 18px;">Examples :</span></strong></p>
<pre><span style="font-size: 18px;"><strong>Input: </strong>a[] = [1, 5, 10, 20, 40, 80], b[] = [6, 7, 20, 80, 100], c[] = [3, 4, 15, 20, 30, 70, 80, 120]
<strong>Output:</strong> [20, 80]
<strong>Explanation:</strong> The elements 20 and 80 appear in all three arrays a, b, and c, making them the only common elements, so the output is [20, 80].</span></pre>
<pre><span style="font-size: 14pt;"><strong>Input: </strong>a[] = [1, 2, 3, 4, 5], b[] = [6, 7], c[] = [8, 9, 10]
<strong>Output:</strong> []
<strong>Explanation:</strong> Since none of the elements in arrays a, b, and c appear in all three arrays, there are no common elements, so the output is [].</span></pre>
