<h2><a href="https://www.geeksforgeeks.org/problems/length-of-longest-cycle-in-a-graph/1">Length of Longest Cycle in a Graph</a></h2><h3>Difficulty Level : Difficulty: Hard</h3><hr><div class="problems_problem_content__Xm_eO"><p><span style="font-size: 14pt;">Given an directed graph with <strong>V&nbsp;</strong>vertices numbered from 0 to V-1 and&nbsp;<strong>E</strong>&nbsp;edges, represented as a 2D array&nbsp;<strong>edges[][]</strong>, where each entry&nbsp;<strong>edges[i] = [u, v]</strong>&nbsp;denotes an edge between vertices&nbsp;<strong>u</strong>&nbsp;and&nbsp;<strong>v</strong>. Each node has at most one outgoing edge.</span></p>
<p><span style="font-size: 14pt;">Your task is to find the length of the&nbsp;<strong>longest cycle&nbsp;</strong>present in the graph. If no cycle exists, <strong>return&nbsp;</strong><strong>-1.</strong></span></p>
<p><span style="font-size: 14pt;"><strong>Note:</strong> A cycle is a path that starts and ends at the same vertex.</span></p>
<p><span style="font-size: 14pt;"><strong>Examples :</strong></span></p>
<pre><span style="font-size: 14pt;"><strong>Input:</strong> V = 7, edges[][] = [[0, 5], [1, 0], [2, 4], [3, 1], [4, 6], [5, 6], [6, 3]]<br><img src="https://media.geeksforgeeks.org/img-practice/prod/addEditProblem/problem_desc/Web/Other/blobid0_1761654225.webp" width="342" height="160"><br><strong>Output: </strong>5<strong>
Explanation: </strong>longest Cycle is 0-&gt;5-&gt;6-&gt;3-&gt;1-&gt;0</span></pre>
<pre><span style="font-size: 14pt;"><strong>Input: </strong>V = 8,<strong> </strong>edges[][] <strong>= </strong>[[0, 1], [1, 2], [2, 3], [3, 0], [4, 1], [5, 4], [6, 2], [7, 6]]<br><img src="https://media.geeksforgeeks.org/img-practice/prod/addEditProblem/problem_desc/Web/Other/blobid1_1761654282.webp" width="345" height="161"><br><strong>Output: </strong>4<br><strong>Explanation: </strong>longest Cycle is 0-&gt;1-&gt;2-&gt;3-&gt;0</span></pre>
