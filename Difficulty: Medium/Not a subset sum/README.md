<h2><a href="https://www.geeksforgeeks.org/problems/smallest-number-subset1220/1">Not a subset sum</a></h2><h3>Difficulty Level : Difficulty: Medium</h3><hr><div class="problems_problem_content__Xm_eO" style="--text-color: var(--problem-text-color);"><p><span style="font-size: 18px;">Given a array <strong>arr[]</strong> of positive integers, find the <strong>smallest positive integer</strong> such that it cannot be represented as the sum of elements of any subset of the given array set.</span></p>
<p><span style="font-size: 18px;"><strong>Examples:</strong></span></p>
<pre><span style="font-size: 18px;"><strong>Input: </strong>arr[] = [3, 1, 2]
<strong>Output:</strong> 7
<strong>Explanation:</strong> 7 is the smallest positive number for which no subset is there with sum 7.
</span></pre>
<pre><span style="font-size: 18px;"><strong>Input: </strong>arr[] = [3, 10, 9, 6, 20, 28]
<strong>Output:</strong> 1
<strong>Explanation:</strong> 1 is the smallest positive number for which no subset is there with sum 1.
</span></pre>
